# libraryproject
[[ 전국 도서관 웹 공간정보시스템 ]]

기간 : 2021.11.24 ~ 2021.12.15

개발 인원 : 1명

개발 플랫폼 : Window8

개발 툴 : Eclipse, Apache-tomcat, exERD, Geocoding, QGIS, PostgreSQL, PostGIS, Geoserver, OpenLayers

사용 언어 : JAVA(JDK 15.0.2), HTML, CSS, JavaScript, JSP/Servlet

사용 언어 : JSTL, GSON, JQuery
